# Dashboard de Ventas Comerciales 2018-2019
## SQL Server · Power Query · Power BI · DAX

## Descripción
Análisis comercial del comportamiento de ventas de una empresa entre 2018 y 2019. El proyecto abarca el flujo completo de datos: desde la base de datos hasta el dashboard interactivo.

## Preguntas que responde el dashboard
- ¿Cómo evolucionaron los ingresos a lo largo del tiempo?
- ¿Qué zonas concentran la mayor cantidad de ventas?
- ¿Qué productos generan mayor impacto en los ingresos?
- ¿Qué productos presentan bajo rendimiento?
- ¿Qué clientes y vendedores mantuvieron un rendimiento constante?

## Herramientas
- SQL Server — modelado y consultas
- Power Query — transformación y limpieza
- Power BI — visualización y DAX
- Esquema estrella con 5 tablas relacionadas

## Estructura del proyecto
- /sql → queries utilizadas
- /pbix → archivo Power BI
- /img → capturas del dashboard
